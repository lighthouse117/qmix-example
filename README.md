# qmix-example

マルチエージェント深層強化学習アルゴリズム QMIX のシンプルな問題への適用

## ライブラリ

- 深層学習フレームワーク : PyTorch
- 実験管理ツール : Sacred

## 論文

[QMIX: Monotonic Value Function Factorisation for Deep Multi-Agent Reinforcement Learning](https://arxiv.org/abs/1803.11485) (
オックスフォード大学, 2018)
